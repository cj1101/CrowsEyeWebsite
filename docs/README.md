## 🛠️ Development Environment

### Requirements
- Node.js 18+ 
- Python 3.8+
- PostgreSQL/SQLite
- Google Cloud SDK (for production)

### Local Development
1. Backend API: Port 8001
2. Frontend: Port 3000
3. Database: SQLite (local) / PostgreSQL (production)
